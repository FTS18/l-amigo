Installation

Get started in under 2 minutes.

FROM CHROME WEB STORE

  1. Visit the L'Amigo Chrome Web Store page
  2. Click "Add to Chrome"
  3. Confirm by clicking "Add extension"
  4. The extension icon appears in your toolbar

FROM SOURCE

git clone https://github.com/yourusername/lamigo.git
cd lamigo
npm install
npm run build

# Load in Chrome:
# 1. Go to chrome://extensions/
# 2. Enable "Developer mode"
# 3. Click "Load unpacked"
# 4. Select the dist folder

REQUIREMENTS: Chrome 88+ or Chromium-based browser

Getting Started

INITIAL SETUP

  1. Click the L'Amigo icon in your toolbar
  2. Click "Get Started"
  3. Enter your LeetCode username
  4. Choose theme preference
  5. Click "Save & Continue"

INTERFACE

  • Friends Tab: View tracked friends with stats
  • Compare Tab: Head-to-head comparison
  • Settings Tab: Configure sync and data

Troubleshooting

POPUP WON'T OPEN

  1. Check extension enabled at chrome://extensions/
  2. Reload extension
  3. Clear browser cache

FRIEND NOT LOADING

  • Invalid username
  • Private profile
  • Rate limited - wait a few minutes

GITHUB SYNC FAILING

"BAD CREDENTIALS"

Token invalid or expired. Generate new token.

"NOT FOUND" (404)

Wrong repo name or no permission.

SYNC STUCK

First sync of 500+ problems takes 5-10 minutes. Don't close popup.

STILL STUCK?

Open a GitHub issue

FAQ

IS IT FREE?

Yes. Open source. MIT License. No ads.

WORKS WITH PREMIUM?

Yes. Premium problems sync normally.

TRACK PRIVATE PROFILES?

No. Only public profiles accessible via API.

REFRESH FREQUENCY?

  • Manual: Anytime
  • Auto background: Every 4 hours
  • Real-time: On "Accepted" verdict

DATA SENT ANYWHERE?

No. All local. Only LeetCode and GitHub APIs contacted.

WILL I GET BANNED?

No. Uses public API with rate limiting.

PRIVATE GITHUB REPOS?

Yes. Works with repo scope token.

FIREFOX VERSION?

Not yet. Contributions welcome.
