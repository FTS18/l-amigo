Features

Everything you need to track and compete with your coding squad

50 FRIENDS

Track up to 50 friends. See streaks, stats, and velocity at a glance.

COMPARE

Side-by-side comparison. Charts for difficulty, topics, languages.

GITHUB SYNC

Auto-push accepted solutions to GitHub. Organized by difficulty.

AUTO DETECT

Real-time "Accepted" detection. Syncs instantly to GitHub.

365 DAYS

Full year of submission history. Accurate streaks and trends.

DISCOVER

Find problems your friends solved that you haven't.

How It Works

Three steps to get started

01

INSTALL

Add from Chrome Web Store. Enter your LeetCode username. Optionally add GitHub token.

02

ADD FRIENDS

Enter LeetCode usernames. Profiles load instantly. Track up to 50 friends.

03

COMPETE

View friend cards with stats. Compare head-to-head with interactive charts.
